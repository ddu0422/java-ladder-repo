# java-ladder
사다리타기 미션 저장소

## 기능 목록 정리 1단계
* 사람이름 입력받기
    * 5자 이하
    * "", null
* 사다리 높이 입력받기
    * 문자
    * 음수
* 사다리 출력
    * 사다리의 라인이 겹치면 안됨

## 기능 목록 정리 2단계
* 실행결과 입력
    * [예외] 참여자 수와 다른 경우
* 사다리 게임 실행
    * 사용자 이동
* 결과를 보고 싶은 참여자 입력
    * All 입력 시 종료
    * [예외] 참여자 이름이 없는 경우
