# 문자열 덧셈

## Todo
* 문자열을 전달하는 경우 구분자를 기준으로 분리한 각 수자의 합을 반환
    * "1" -> 1
    * "1,2" 들어오면 3을 반환
    * "1:2" -> 3
    * "//;\n1;2;3" -> 6
    
* 숫자 이외의 값 또는 음수를 전달하는 경우 예외
    * -1,2,3
    * a,b,c
